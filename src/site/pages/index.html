Make Static Sites Great Again

Static Site Generator

A pre-built template project combining Metalsmith, Gulp and Tailwindcss. Quick start your next static website using the combination of 3 great technologies.

GitHub

Commands

Use the following commands to get started.

// To Install Dev Depedencies
npm install

// To start development and server for live preview
npm run dev

// To generate minifed,optamized files for live server
npm run build

Options

To change the path of files and destination/build folder or font weights check package.json file

"options": {
  "config": {
    "tailwindjs": "./tailwind.js"
  },
  "paths": {
    "root": "./",
    "src": {
      "base": "./src",
      "css": "./src/css",
      "js": "./src/js",
      "img": "./src/img"
    },
    "dist": {
      "base": "./dist",
      "css": "./dist/css",
      "js": "./dist/js",
      "img": "./dist/img"
    },
    "build": {
      "base": "./build"
    }
  }
}
